# Firmware Build and Test Workflow

Required tools
- Node.js 20+
- PlatformIO

Always build web UI first
- npm ci
- npm run build

Tests
- npm test
- Note: Windows file locks can cause EBUSY when deleting html_*.h files. Close any process that might open them and retry.

Hardware build (required)
- pio run -e esp32dev

ESP32-S3 build and flash
- pio run -e esp32s3dev_16MB_opi_can --target upload

Generated files
- wled00/html_*.h are generated by the build script and must not be edited by hand.

Do not skip
- Run npm test
- Run a hardware build at least once (esp32dev)
